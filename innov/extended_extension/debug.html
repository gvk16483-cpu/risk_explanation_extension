Email Reader Extension - Debug Console

📋 How to Debug:
  1. Open Gmail and select an email
  2. Open Chrome DevTools (F12)
  3. Go to the "Console" tab
  4. You should see logged email data like: 
    Extracted email data: {
  subject: "...",
  from: "...",
  fromEmail: "...",
  to: "...",
  body: "..."
}
  5. Check if fromEmail is being extracted correctly
🔍 What to Check:
  • Sender Email: Should show like "john@example.com"
  • Body: Should include all text and URLs as plain text
  • Links: URLs should appear naturally in the body text
Steps to Reload Extension:
  1. Go to chrome://extensions/
  2. Find "Email Reader" extension
  3. Click the Refresh button
  4. Go back to Gmail and open an email
  5. Click the Email Reader extension icon
